- 可变参数模板
  这意味着你可以在声明之后，把需要使用的各种类型的参数都丢进去，并且不用考虑重载这类事情。

### 关于使用  
- 尽量不要使用`LinkNode<Args...>`  
  它是这个数据结构实现的基础  
  对数据包的大小需要手动输入  
  如果不小心，容易出一些事故
- 不要把会动态分配内存的类型放进去
